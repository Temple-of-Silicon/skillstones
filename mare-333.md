| Height | Front | Central | Back |

| Close-mid | e /e/ | - | o /o/ |
| Open-mid | è /ɛ/ | - | - |

### Romanization Key

Special romanizations:
- ng = /ŋ/
- è = /ɛ/

### Syllable Structure

**Formula:** (C)V(C)(C)

Variable complexity

- Maximum onset: 1 consonant(s)
- Maximum coda: 2 consonant(s)

### Phonological Rules

**Nasal Assimilation:** /n/ assimilates to place of following stop
  Rule: `n → m / _[+labial], n → ŋ / _[+velar]`

**Epenthesis:** Insert i to break illegal clusters
  Rule: `∅ → i / C_C (when cluster violates phonotactics)`



## §1.1 Prosody

### Summary

lexical stress, syllable-timed rhythm

### Stress System

**Type:** Lexical (unpredictable) stress

**Primary Stress:** Stress is lexically specified (unpredictable)
**Secondary Stress:** Initial if primary is not initial

#### Stress Effects

- **Vowel reduction:** Unstressed vowels reduce to schwa or centralize
- **Stressed vowel lengthening:** Stressed vowels are phonetically longer

### Rhythm Type

**Syllable-timed (isochronous syllables)**

- Equal time for each syllable
- Little vowel reduction
- Clear syllable boundaries
- Relatively even rhythm

*Similar to: Spanish, French, Yoruba*

### Intonation Patterns

| Type | Contour | Example |
|------|---------|---------|
| statement | level-low | The cat sat on the mat. ↘ |
| yes-no question | high-rising | Is the cat on the mat? ↗ |
| wh-question | high-falling | Where is the cat? ↘ |
| continuation | sustained | When I got home, → ... |
| focus | pitch-peak | THE CAT sat on the mat. |
