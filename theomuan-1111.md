**Formula:** (C)(C)V(C)

(C)(C)V(C) (moderate complexity)

### Phonological Rules

**Nasal Assimilation:** /n/ assimilates to place of following stop
  Rule: `n → m / _[+labial], n → ŋ / _[+velar]`

**Vowel Harmony:** Vowels agree in height within a word
  Rule: `V → [αheight] / V[αheight]___`

**Epenthesis:** Insert i to break illegal clusters
  Rule: `∅ → i / C_C (when cluster violates phonotactics)`



## §1.1 Prosody

### Summary

fixed stress, stress-timed rhythm

### Stress System

**Type:** Fixed initial stress

**Primary Stress:** Always on initial syllable
**Position:** initial
**Secondary Stress:** On heavy syllables before primary

#### Stress Effects

- **Vowel reduction:** Unstressed vowels reduce to schwa or centralize

### Rhythm Type

**Stress-timed (isochronous stresses)**

- Equal time between stressed syllables
- Unstressed syllables compressed
- Variable syllable length
- Strong vowel reduction

*Similar to: English, German, Russian*

### Intonation Patterns

| Type | Contour | Example |
|------|---------|---------|
| statement | level-low | The cat sat on the mat. ↘ |
| yes-no question | rising | Is the cat on the mat? ↗ |
| wh-question | falling | Where is the cat? ↘ |
| continuation | sustained | When I got home, → ... |
