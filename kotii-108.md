### Vowels

| Height | Front | Central | Back |
|--------|-------|---------|------|
| Close | i /i/ | - | u /u/ |
| Close-mid | e /e/ | - | o /o/ |
| Open | - | a /a/ | - |

### Romanization Key

Special romanizations:
- gh = /ɣ/
- zh = /ʒ/
- ng = /ŋ/
- kh = /x/
- ch = /tʃ/
- sh = /ʃ/

### Syllable Structure

**Formula:** (C)V(C)(C)

Variable complexity

- Maximum onset: 1 consonant(s)
- Maximum coda: 2 consonant(s)

### Phonological Rules

**Nasal Assimilation:** /n/ assimilates to place of following stop
  Rule: `n → m / _[+labial], n → ŋ / _[+velar]`

**Final Devoicing:** Voiced obstruents become voiceless word-finally
  Rule: `[+voiced, -sonorant] → [-voiced] / _#`

**Epenthesis:** Insert i to break illegal clusters
  Rule: `∅ → i / C_C (when cluster violates phonotactics)`



## §1.1 Prosody

### Summary

weight-sensitive stress, stress-timed rhythm

### Stress System

**Type:** Weight-sensitive stress system

**Primary Stress:** Falls on heaviest syllable within stress window
**Secondary Stress:** Initial if primary is not initial

#### Stress Effects

- **Stressed vowel lengthening:** Stressed vowels are phonetically longer

### Rhythm Type

**Stress-timed (isochronous stresses)**

- Equal time between stressed syllables
- Unstressed syllables compressed
- Variable syllable length
- Strong vowel reduction

*Similar to: English, German, Russian*

### Intonation Patterns

| Type | Contour | Example |
|------|---------|---------|
| statement | level-low | The cat sat on the mat. ↘ |
| yes-no question | fall-rise | Is the cat on the mat? ↗ |
| wh-question | high-falling | Where is the cat? ↘ |
| continuation | level | When I got home, → ... |
| focus | high-pitch-accent | THE CAT sat on the mat. |

## §3 Syntax

### Phrase Structure

**Noun Phrases:**
- Adjective position: after noun
- Genitive position: after noun
- Determiner position: typically before noun

**Example NP structure:** (DET) NOUN (ADJ)

### Questions

**Yes/No Questions:** Add question particle **nunghshesh** at end of sentence.

**Content Questions:** Question word appears in situ (in the position of the questioned element).

Negation via suffix

### Subordination

Relative clauses: follow the head noun.
Complement clauses: typically follow the main verb.
