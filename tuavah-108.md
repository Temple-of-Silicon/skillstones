### Vowels

| Height | Front | Central | Back |
|--------|-------|---------|------|
| Close | i /i/ | - | u /u/ |
| Close-mid | e /e/ | - | o /o/ |
| Open | - | a /a/ | - |

### Romanization Key

Special romanizations:
- ng = /ŋ/
- sh = /ʃ/
- ch = /tʃ/

### Syllable Structure

**Formula:** (C)(C)V

Variable complexity

- Maximum onset: 2 consonant(s)
- Maximum coda: 0 consonant(s)

### Phonological Rules

**Nasal Assimilation:** /n/ assimilates to place of following stop
  Rule: `n → m / _[+labial], n → ŋ / _[+velar]`

**Palatalization:** Velars become palatals before front vowels
  Rule: `k → tʃ / _[+front], g → dʒ / _[+front]`

**Epenthesis:** Insert i to break illegal clusters
  Rule: `∅ → i / C_C (when cluster violates phonotactics)`



## §1.1 Prosody

### Summary

morphological stress, syllable-timed rhythm

**Type:** Morphologically-conditioned stress

**Primary Stress:** Stress position depends on morphological structure

- Little vowel reduction
- Clear syllable boundaries
